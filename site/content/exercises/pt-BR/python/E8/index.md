Variáveis ​​podem armazenar dados de diferentes tipos — até agora, discutimos números, strings, listas e booleanos. Você pode determinar o tipo de uma variável usando `type()`:


```python
type(45)
```
*Output:*
```python
int
```

Aqui está o tipo de string:
```python
type("Hello")
```
*Output:*
```python
str
```

O tipo de lista:
```python
type([2, 5, 7, 10])
```
*Output:*
```python
list
```

O tipo booleano:
```python
type(True)
```
*Output:*
```python
bool
```
